Question key rules

Format
- {SURVEY_TYPE}_{FIELD}

Rules
- SURVEY_TYPE uses enum name: BASIC or DETAILED.
- FIELD is an upper snake case slug that is stable over time.
- Use short semantic names from the question meaning, not the text wording.

Examples
- BASIC_HEIGHT
- BASIC_WEIGHT
- BASIC_BLOOD_PRESSURE_STATUS
- DETAILED_SLEEP_QUALITY
- DETAILED_EXERCISE_FREQUENCY
